# JSON Viewer - Elegant JSON Visualizer

A modern and elegant web application for viewing, formatting, and manipulating JSON data. Built with React and Tailwind CSS, it offers an intuitive and responsive interface for working with JSON efficiently.

## 🚀 Features

### ✨ JSON Manipulation
- **Beautify**: Format JSON with proper indentation for better readability
- **Minify**: Remove unnecessary whitespace to compress JSON
- **Real-time validation**: Instantly detect syntax errors with detailed messages
- **Syntax highlighting**: Visual highlighting of JSON syntax in the editor

### 📊 Visualizations
- **Text editor**: Clean interface for direct JSON editing
- **Tree View**: Interactive hierarchical visualization with expand/collapse nodes
- **Mode toggle**: Easy switching between editor and tree view

### 🛠️ Utilities
- **Copy to clipboard**: Copy current JSON to clipboard
- **Download**: Download JSON as a `.json` file
- **History**: Keep the last 10 manipulated JSONs (persisted in localStorage)
- **Clear**: Complete editor reset
- **Statistics**: Real-time line and character counter
- **Dark Mode**: Optional dark theme for better visual experience

### 🎨 Interface
- **Modern design**: Clean and elegant interface with Tailwind CSS
- **Responsive**: Works perfectly on desktop, tablet, and mobile
- **Smooth animations**: Polished transitions and visual effects
- **Accessibility**: Keyboard navigation and clear visual indicators

## 🚀 How to Use

### Installation
1. Clone or download this repository
2. Open the `index.html` file in any modern browser
3. No need to install dependencies - everything works via CDN!

### Basic Usage
1. **Paste your JSON**: Paste JSON in the editor text area
2. **Automatic validation**: The system will automatically validate the syntax
3. **Formatting**: Use "Beautify" or "Minify" buttons to format
4. **Visualization**: Switch between "Editor" and "Tree View" for different perspectives
5. **History**: Access history by clicking the clock icon
6. **Download**: Download formatted JSON as a file

### Shortcuts and Tips
- **Dark Mode**: Click the sun/moon icon in the header
- **History**: Click the clock icon to view previous JSONs
- **Statistics**: See line and character counters in the toolbar
- **Responsive**: On smaller screens, the layout adapts automatically

## 🛠️ Technologies Used

- **React 18**: JavaScript framework for reactive interface
- **Tailwind CSS**: CSS framework for modern styling
- **Babel Standalone**: JSX compilation in the browser
- **Lucide Icons**: Modern and consistent icons
- **Prism.js**: Syntax highlighting for JSON
- **LocalStorage**: Persistence of history and preferences

## 🎯 Technical Features

### Performance
- **Zero build**: Works directly in the browser without a build process
- **Optimized CDN**: Fast loading via reliable CDNs
- **Lazy loading**: Components loaded on demand
- **Debounced validation**: Optimized validation to avoid overloading the browser

### Security
- **Client-side only**: All data remains in your browser
- **No server**: No data sent to external servers
- **Secure LocalStorage**: Data saved locally with browser encryption

### Accessibility
- **Keyboard navigation**: Full support for mouse-free navigation
- **Adequate contrast**: Colors optimized for readability
- **Visual indicators**: Clear feedback for all actions
- **Responsive design**: Automatic adaptation to different screen sizes
